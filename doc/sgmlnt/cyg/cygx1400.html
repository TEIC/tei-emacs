Install the DocBook stylesheets

  1. Unzip the archive

    Extract the archive containing the DocBook stylesheets into a new stylesheets subdirectory, e.g. /usr/local/lib/sgml/stylesheets. This will create a new subdirectory docbook. If you already have an older version of the stylesheets in this directory, you may want to rename it to e.g. db154 (according to the release) before extracting the new version. This makes it easier to revert to the previous version if you should need to do so.

  2. Register catalog

    The DTD stylesheets archive contains a catalog file named catalog which must be added to the SGML_CATALOG_FILES environment variable. Use the procedure as described above. Add the CygWin path of the catalog file to SGML_CATALOG_FILES in /etc/profile.

  3. Unzip the documentation

    Unpack the stylesheet documentation archive into a convenient directory, e.g. /usr/local/lib/sgml/doc/dbstylesheets.
