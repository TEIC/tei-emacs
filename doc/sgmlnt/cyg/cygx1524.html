Further Reading

The DocBook DTD documentation is available at OASIS.

Norman Walsh has written DocBook: The Definitive Guide, which is available as a book at O'Reilly and online.

If you want to customize the DocBook stylesheets, you don't want to miss the DocBook stylesheet documentation. These documents also contain a very clear and concise description of the index creation process with the collateindex.pl script, see the file indexing.htm.

An excellent online-resource for HTML coding is SELFHTML. Unfortunately the document is only available in German, but an english translation is on the way.

The Text Encoding Initiative has published a DTD which seems more geared towards linguistic applications. There is also a somewhat stripped-down version called TEI Lite. Richard Light (with some additions by Jon Bosak) has published TEI Lite DSSSL print stylesheets.

DSSSL stylesheets for HTML come in handy if you are not satisfied with what your browser prints. Stylesheets are available for HTML 3.2.
