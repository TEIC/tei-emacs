The first steps with the HTML DTDs

Let us now create a new HTML document. Type C-x C-f ~/htmltest.html and insert a formal public identifier via the DTD menu. Now use the Markup->Insert Element menu commands and related commands to build a HTML document. You can preview the document in your browser while editing. Don't forget to hit the Refresh button of your browser after changes in the source code. Use the SGML->Validate menu command to validate your document against the DTD.
