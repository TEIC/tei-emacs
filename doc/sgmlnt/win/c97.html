This chapter briefly introduces the components that make up the complete system. The following chapters provide detailed installation instructions, and each chapter contains the exact download locations for the components.

Edit texts

Obviously we'll have to write or edit some kind of text if we want to create and publish SGML documents. We will use the NT port of GNU Emacs. This is one of the most remarkable and versatile pieces of software, which can do anything you may think of except maybe preparing coffee. We will adapt and use it to edit SGML source documents (using Lennart Staflin's PSGML mode), SGML document type definitions (using Tony Graham's TDTD mode), DSSSL stylesheets, and of course the editor's own configuration files. Moreover, we will use it as a kind of IDE by turning it into a frontend for the command-line utilities which do much of the work behind the scenes.
