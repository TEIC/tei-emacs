System Requirements

The software described below will run on Windows NT or Windows 95/98/ME. Installation on Windows 95/98/ME needs a few modifications of the installation procedure. This will be pointed out where necessary. The installed software occupies about 70 MB hard disk space. This may be considerably more if you use a FAT file system on a large harddisk instead of NTFS due to larger cluster sizes (consider partitioning the disk in this case). The memory size and CPU speed requirements are not extraordinary, I once ran the system on a Pentium 100 with 32 MB RAM and Windows NT 4.0 SP3 without problems. But I have to admit that especially OpenJade benefits noticeably from more RAM.
